Hexo 的草稿模式

0x0
2017-03-22

编写草稿

在创建文章的时候，只要指定文章的模板是draft，就可以将文章创建到source/_drafts/目录下

放在_drafts目录中的草稿，使用hexo g命令生成静态文件的时候，草稿不会生成

查看草稿

如果想要查看草稿的效果，可以在启动本地服务的时候增加一个参数：hexo s –draft，或者在 _config.yml文件中进行配置：每次启动本地服务的时候，都会渲染编写的草稿
